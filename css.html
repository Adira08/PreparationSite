1. Difference between flex row and column.

The flex is a CSS property which sets the flexible length on flexible items.
The flex-direction is flex property which specifies the direction of the flexible items.
     1. Flex-direction: row - This property helps to flexible items are displayed horizontally, as a row.
     2. Flex-direction: column - This property helps to flexible items are displayed virtically, as a column.
